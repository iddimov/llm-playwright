# OrangeHRM Features

This document outlines the core functional features of the OrangeHRM application that are within the scope of our automated testing framework.

## 1. Authentication (Login)

The application provides a secure login portal for administrators and employees.

*   **Valid Login**: Users with correct credentials must be able to log in successfully and be directed to the dashboard.

## 2. Personnel Information Management (PIM)

The PIM module is the core feature for managing employee records within the organization.

*   **Employee Addition**: Administrators or HR personnel can add new employees to the system. This involves entering basic details such as the employee's First Name and Last Name.
*   **Employee Verification/Listing**: After an employee is added, their record should be visible and verifiable within the system's employee list or through their specific profile view.
